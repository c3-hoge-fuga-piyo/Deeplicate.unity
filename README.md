# Deeplicate

[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat-square)](http://c3.mit-license.org/2016/)

アセットをディレクトリ単位でコピーした際に、コピー元のアセットを参照してしまうお悩みを解決するヤーツ

## Requirement

Unity 5.4 or later

## Usage

1. コピーしたいディレクトリを選択し「Assets/Deeplicate」を実行する
2. 以上
